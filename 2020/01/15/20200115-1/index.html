[KR] 기초 통계학 정리 pt.1 (표본? 샘플링?)

Posted on 2020-01-15 Edited on 2020-08-02 In Statistics

평소에 수학에 대해서는 스스로가 많이 부족하다고 인지하고 있는 반면, 통계는 어중간하게 알고 있다는 느낌을 받을 때가 많습니다. 어쩌면 그럴 때가 더 위험할 수도 있을 거 같다는 생각에 정리를 해보고자 합니다.

통계학의 목적

  • 기술통계 - 데이터 전체를 일일이 다 들여볼 수 없으니까 데이터를 잘 설명하기 위한 통계

  • 추측통계 - 가지고 있지 않은 임의의 데이터를 기존의 데이터를 기반으로 예측하기 위한 통계


모집단과 표본

  • 모집단(population) - 아직 가지고 있지 않은 모르는 데이터를 포함한 전-체 데이터

  • 표본(Sample) - 현재 확보한 데이터

  • 확률변수(Random variable) - 확률적 법칙에 따라 변화하는 값

    • 무작위 실험을 했을 때, 특정 확률로 발생하는 각 결과를 수치적 값으로 표현하는 변수
    • 어떠한 확률이 정해졌을 때, 혹은 확률이 변화할 때 → 거기에 따라 함께 변화하는 값. 확률변수


  • 실현값 - 확률변수의 구체적인 값
    • 예) 확률에 따라 확률변수를 구했을 때.


  • 샘플링(Sampling : 표본추출) - 모집단에서 표본을 얻는 과정.
    • 단순랜덤샘플링(simple random sampling) - 모집단에서 하나하나 요소를 무작위로 선택하는 방법


  • 샘플사이즈 - 표본의 크기나 가지고 있는 데이터의 수
    • 크다 작다 (o)
    • 많다 적다 (x)


  • 전수조사 - 모집단 전체를 조사하는 것
    • 매우 드문 일이며, 보통 샘플만 가지고 모집단을 추정함.


  • 표본조사 - 모집단의 일부만 조사하는 것




확률이라는 추상화된 개념

  • 확률(probability) - 사상(사건, 부분집합, event)을 입력하면 숫자(확률값)가 출력되는 함수
    • X = 데이터일 때, 확률은 $P(X)$ 라고 표현할 수 있음


  • 확률분포(probability distribution) - 확률변수와 그 값이 나올 수 있는 확률을 대응한 것.
    • 확률변수의 모든 값과, 그에 대응하는 확률들이 어떻게 분포하고 있는지 나타냄.


  • 확률분포를 따르다 (follow probability distribution) - 어떤 데이터가 어떤 확률분포에 대응할 때 확률분포를 따른다 고 함.

  • 모집단분포 - 모집단이 따르는 확률분포

  • 무한모집단 - 무한한 크기를 가진 모집단 (예: 주사위를 던지는 횟수)
